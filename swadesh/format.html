Each line in each .txt file has these tab-separated columns.
word
lemma, if available
POS tag, from Google tag set
transliteration into Roman alphabet
mapping into IPA
translation into English
probability or confidence, 0.0 to 1.0, if available
dialect, if known
topic/domain keyword, if available
comma-separated source(s) (pointer to data source(s))
comma-separated list of related morphological variants
